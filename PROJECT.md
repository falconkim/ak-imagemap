# 프로젝트: 데스크톱 이미지맵 편집기 (Image Map Editor)

본 프로젝트는 웹 표준 이미지맵([map], [area] 태그) 코드를 시각적으로 디자인하고 실시간으로 추출할 수 있는 Electron 기반의 데스크톱 애플리케이션입니다. 외부 프레임워크 없이 pure HTML/CSS/JavaScript(Vanilla JS) 환경에서 개발을 진행하며, 가볍고 직관적인 UI/UX를 제공합니다.

---

## 1. 프로젝트 개요

* **개발 환경:** Node.js, Electron, Vite
* **프론트엔드:** HTML5, Pure CSS, Vanilla JavaScript
* **타깃 플랫폼:** Windows (설치형/포터블 앱)

---

## 2. 주요 기능 명세

### 2.1 이미지 관리 및 네트워크 처리

* **로컬 이미지 불러오기:** 탐색기를 통해 로컬 이미지 파일(Web 지원 포맷) 로드
* **온라인 URL 불러오기:** [http://], [https://] 경로의 이미지 주소 입력 및 로드
* **지원 확장자:** Web 표준 포맷 (PNG, JPG/JPEG, GIF, WEBP, SVG)
* **네트워크 에러 핸들링:** * 온라인 이미지 호출 실패 시 상세 에러 원인 분석 후 얼럿 또는 상태바에 메시지 출력 (예: 404 Not Found, CORS 제한, 500 서버 에러, 네트워크 연결 유실 등)

### 2.2 이미지 뷰어 (좌측/중앙 영역)

* **마우스 휠/단축키 기반 확대/축소 (Zoom In/Out):** 10% ~ 50% 단위 제어, 최대 400% 확대 및 최적 맞춤 기능
* **캔버스 스크롤 및 핸드 툴:** 대형 이미지 작업 시 마우스 드래그(Space + Drag) 또는 스크롤바를 통한 자유로운 이동

### 2.3 이미지맵 드로잉 및 편집 (중앙 작업 영역)

* **도형 그리기 기능:** 웹 표준을 지원하는 3가지 영역 형태 제공
* **사각형 (rect):** 시작점 클릭 후 드래그하여 영역 설정
* **원 (circle):** 중심점 클릭 후 반지름 드래그 설정
* **다각형 (poly):** 정점(Vertex)을 연속 클릭하여 다각형 생성, 더블 클릭으로 닫기


* **상태 관리 및 수정 기능 (정밀 제어):**
* **이동 (Move):** 그려진 영역을 마우스로 드래그하여 위치 이동
* **크기 변경 (Resize):** 각 영역의 꼭짓점/바운딩 박스 조절 핸들을 제공하여 크기 및 형태 변형
* **요소 선택 및 삭제:** 캔버스 내부 요소 클릭 또는 레이어 리스트에서 선택 후 [Delete] 키로 삭제
* **[기능 확장] Undo / Redo:** 실수 방지를 위한 작업 히스토리(되돌리기/다시실행) 기능 (최대 30개)



### 2.4 속성 및 코드 실시간 연동 (우측 영역)

* **하이퍼링크 및 속성 설정:**
* 각 영역 선택 시 [href], [target](_blank, _self 등), [alt] 속성 변경 창 제공


* **실시간 HTML 소스코드 반영:**
* 캔버스에서 영역을 그리거나 수정할 때마다 우측 코드 에디터 창에 [map], [area] 태그 코드가 실시간 텍스트로 갱신


* **[기능 확장] Map ID 커스텀 설정:** 사용자가 원하는 `usemap` 이름 및 `id` 속성을 상단 또는 우측 패널에서 직접 지정 가능

### 2.5 최종 저장 및 내보내기

* **HTML 파일 저장:** [img usemap="..."] 및 [map] 코드가 온전히 포함된 HTML 파일(.html) 내보내기 기능
* **[기능 확장] 프로젝트 파일(JSON) 내보내기/불러오기:** 작업 중인 이미지 경로와 맵 좌표 데이터를 고유 프로젝트 파일로 저장하여, 추후 재수정 및 이어 그리기 지원

## 3. 기능 확장 상세 (완벽한 편집기를 위한 추가 제어)

완벽한 이미지맵 편집기 구현을 위해 기존 요구사항 외에 아래 4가지 핵심 기능을 추가 확장합니다.

1. **레이어 리스트 패널 (오른쪽/우측 배치):**
* 현재 그려진 영역들을 순서대로 리스트화하여 보여줍니다. (예: [Area 1 - 사각형], [Area 2 - 다각형])
* 복잡하게 겹쳐 있는 영역을 명확히 선택하고, 레이어 이름을 더블클릭하여 직관적으로 구별할 수 있게 합니다.


2. **그리드(Grid) 및 스냅(Snap) 기능:**
* 정밀한 좌표 픽셀 매칭을 위해 바둑판 모양의 그리드 라인 토글 기능을 제공합니다.
* 그리드 선에 좌표가 딱 붙는 '스냅 기능'을 켜고 끌 수 있게 하여 작업 생산성을 높입니다.


3. **키보드 방향키 미세 조정 (Nudge):**
* 선택된 영역이나 정점을 키보드 방향키([↑], [↓], [←], [→])로 1px씩 미세하게 이동할 수 있도록 제어 기능을 강화합니다. (Shift 조합 시 10px 이동)


4. **반응형 이미지맵 지원 스크립트 선택 옵션:**
* 고정 픽셀 방식의 이미지맵은 브라우저 크기가 변할 때 좌표가 깨집니다.
* 내보내기 시 RWD(반응형 크기 조절)를 지원하는 가벼운 JavaScript 모듈 코드를 하단에 선택적으로 포함하여 저장할 수 있는 옵션을 제공합니다.

## 4. UI/UX 레이아웃 구조

앱은 상단 툴바와 좌/우/중앙의 3단 분할 레이아웃으로 설계되며, 프레임워크 없는 순수 CSS Flex/Grid 속성을 이용해 유연하게 반응하도록 구현합니다.

```
+------------------------------------------------------------------------------------+
|  [상단 메뉴/툴바]  새 프로젝트 | 이미지 로드 | 사각형 | 원 | 다각형 | 핸드툴 | 확대/축소 | Map ID 입력 |
+------------------------+------------------------------------+----------------------+
| [좌측 내비게이터]        | [중앙 메인 작업 캔버스]            | [우측 속성 및 코드]  |
|                        |                                    |                      |
|  * 이미지 미리보기     |   +----------------------------+   |  * 선택 영역 속성    |
|    (썸네일 형태)       |   |                            |   |    - href, target    |
|                        |   |     불러온 이미지 영역      |   |                      |
|  * 레이어 리스트       |   |                            |   |  * 실시간 HTML 코드  |
|    - 사각형 1          |   |    [o] 그리기/수정 핸들    |   |    <map name="...">  |
|    - 원 1              |   |                            |   |      <area ...>      |
|                        |   +----------------------------+   |    </map>            |
|                        |                                    |                      |
+------------------------+------------------------------------+----------------------+
| [하단 상태바]  이미지 경로: http://... | 마우스 좌표: X: 120, Y: 450 | 상태: 정상           |
+------------------------------------------------------------------------------------+

```

## 5. 프로젝트 디렉터리 구조 (Vite + Electron)

```
image-map-editor/
├── main.js                 # Electron 메인 프로세스 (윈도우 생성, 파일 I/O 시스템 제어)
├── preload.js              # IPC 통신 및 안전한 API 브리지 정의
├── index.html              # 앱의 메인 UI 레이아웃 컨테이너
├── package.json            # 의존성 및 빌드 스크립트 정의
├── vite.config.js          # Vite 빌드 설정 파이프라인
└── src/                    # 프론트엔드 어플리케이션 소스 (Vanilla)
    ├── css/
    │   └── style.css       # 애플리케이션 레이아웃 및 캔버스 스타일 (Pure CSS)
    ├── js/
    │   ├── app.js          # 메인 초기화 및 이벤트 바인딩 컨트롤러
    │   ├── canvas.js       # 드로잉 캔버스 로직 (도형 그리기, 드래그, 수정, 배율 조절)
    │   ├── network.js      # 이미지 URL 요청 및 HTTP Status/에러 핸들러
    │   └── parser.js       # 좌표 데이터를 HTML 코드로 가공 및 실시간 파싱
    └── assets/             # 기본 아이콘 및 이미지 리소스

```

## 6. 개발 단계별 마일스톤

1. **1단계: 환경 구성 및 메인 윈도우 수립**
* Vite와 Electron 환경 융합 및 IPC 통신 파이프라인 구축


2. **2단계: 이미지 로더 및 네트워크 예외 처리 인터페이스**
* 로컬 및 웹 URL 이미지 로딩 기능 및 상세 에러 디버깅창 구현


3. **3단계: 캔버스 마우스 좌표계 코어 개발**
* 줌/인 아웃 상태에서도 왜곡 없는 순수 좌표 연산 및 3대 도형 드로잉 핸들러 구현


4. **4단계: 요소 수정 및 실시간 파서 연동**
* 바운딩 박스를 활용한 정점 이동/크기 변경 스크립트 작성 및 우측 HTML 텍스트 실시간 바인딩


5. **5단계: 데이터 익스포트 및 패키징**
* 완벽한 스크립트가 포함된 HTML 내보내기 검증 및 Electron-Builder를 통한 최종 설치형(.exe) 배포판 생성
