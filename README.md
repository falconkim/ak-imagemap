# AK Image Map Editor (데스크톱 이미지맵 편집기)

웹 표준 이미지맵(`<map>`, `<area>` 태그) 코드를 시각적으로 손쉽게 작성, 수정 및 추출할 수 있는 데스크톱 전용 에디터입니다. 외부 프레임워크를 배제한 순수 HTML/CSS/Vanilla JavaScript 기반으로 제작되어 가볍고 직관적인 UI/UX를 제공합니다.

---

## 💾 다운로드 및 즉시 실행 (Download & Run)

개발 환경을 구축하지 않고도 데스크톱 애플리케이션 설치 파일을 직접 다운로드하여 바로 실행할 수 있습니다.

1. **설치 파일 다운로드:**
   * [GitHub Releases 페이지](https://github.com/falconkim/ak-imagemap/releases)에서 최신 버전의 **`Image Map Editor Setup 1.0.0.exe`** 설치 파일을 다운로드합니다.
   * 또는 이 저장소에 업로드된 [dist-electron/Image Map Editor Setup 1.0.0.exe](https://github.com/falconkim/ak-imagemap/blob/main/dist-electron/Image%20Map%20Editor%20Setup%201.0.0.exe) 링크를 클릭하여 우측의 **Download** 버튼을 눌러 내려받을 수 있습니다.
2. **설치 및 실행:**
   * 다운로드한 `Image Map Editor Setup 1.0.0.exe` 파일을 더블클릭하여 설치를 진행합니다.
   * **자동 업데이트 감지:** 이미 이전 버전의 프로그램이 설치되어 있는 경우, 설치 프로그램이 기존 설치 상태를 자동으로 감지하고 새 버전을 안내하는 확인 창을 표시하여 안전하게 업데이트를 완료해 줍니다.

## ✨ 주요 기능 명세

### 1. 3가지 영역 형태의 시각적 드로잉
* **사각형 (rect):** 드래그하여 두 꼭짓점 좌표 지정
* **원 (circle):** 중심점 지정 후 드래그하여 반지름 설정
* **다각형 (poly):** 클릭을 통해 다중 정점(Vertex)을 연속 생성하고, 더블클릭하여 다각형 폐합

### 2. 라이브 양방향 코드 동기화 (Live HTML Editing)
* **실시간 생성:** 영역을 그릴 때마다 우측 코드 패널에 웹 표준 이미지맵 코드 자동 반영
* **역파싱 반영:** 코드 에디터 패널에서 이미지맵 HTML 태그를 직접 수정(좌표 변경, 영역 추가, 링크/대체 텍스트 변경 등)한 후 **`Ctrl + Enter`** 혹은 적용(▷) 버튼을 누르면 화면의 드로잉 캔버스에 수정사항이 실시간 역반영됨

### 3. 유연한 프로젝트 데이터 가져오기 / 내보내기
* **HTML 내보내기:** 이미지맵 코드가 내장된 HTML 파일 생성
  * *선택 옵션:* 브라우저 크기가 바뀌어도 이미지 크기에 맞춰 좌표가 자동 조정되는 **반응형 이미지맵 스크립트(RWD)** 내장 옵션 제공
* **프로젝트 JSON 내보내기:** 작업 내역(도형 데이터, 링크, 원본 이미지 등)을 온전히 저장하여 추후 이어 그리기 지원
* **프로젝트 불러오기:** 이전에 저장한 프로젝트 `.json`은 물론, 일반 `.html` 이미지맵 파일도 불러와 캔버스로 재구성 가능
  * *스마트 이미지 로더:* HTML 불러오기 시 원격 URL 이미지는 CORS 제약 없이 다운로드하여 로드하고, 로컬 상대 경로는 HTML 파일 기준으로 자동 탐색/로드하며 실패 시 수동 복구 선택지 제공

### 4. 강력한 그리기 보조 및 편집 도구
* **캔버스 Zoom & Pan:** `Ctrl` + 마우스 휠로 화면 확대/축소(10% ~ 400%), 마우스 휠 및 마우스 드래그(Space + Drag)로 자유로운 화면 이동
* **미니맵 내비게이터:** 우측 내비게이터 뷰어를 통해 현재 캔버스 화면의 노출 위치를 시각화하고 클릭/드래그하여 빠른 캔버스 패닝 가능
* **그리드 & 스냅:** 좌표 픽셀을 정밀하게 맞출 수 있는 눈금자 격자(Grid) 활성화 및 좌표 자동 흡착(Snap) 지원
* **미세 조정 (Nudge):** 선택 영역 및 꼭짓점을 키보드 방향키(`↑`, `↓`, `←`, `→`)로 1px씩 미세 이동 (Shift 조합 시 10px 이동)
* **단축키 복제 및 붙여넣기:** 영역 선택 상태에서 `Ctrl + Drag`로 손쉬운 복제, 또는 `Ctrl + C` 및 `Ctrl + V`로 X축 +10px 이동된 영역 복제
* **작업 히스토리 (Undo / Redo):** 실수 방지를 위해 최대 30단계의 실행 취소 및 재실행 지원

## 🛠️ 기술 스택 (Technology Stack)

* **데스크톱 런타임:** Electron (`^42.4.1`)
* **빌드 및 개발 번들러:** Vite (`^8.0.16`)
* **프론트엔드:** HTML5, CSS3 (Glassmorphism & Dark Theme), Pure JavaScript (ES Modules, Vanilla JS)
* **아이콘 리소스:** Remix Icon
* **사용 폰트:** Google Fonts (Outfit, JetBrains Mono)
* **설치본 빌더:** Electron-Builder (`^26.15.3`)

## 🚀 개발 및 실행 가이드

### 1. 의존성 설치
프로젝트의 안전한 패키지 구성을 위해 자동 스크립트 실행을 제한하는 명령어로 의존성을 설치합니다.
```bash
npm install --ignore-scripts
```

### 2. 개발 모드 실행
프론트엔드 로컬 서버(Vite)와 일렉트론 셸을 동시 실행합니다. (GPU 크래시 방지 안전 플래그가 자동 적용됩니다)
* **Vite 개발 서버 구동:**
  ```bash
  npm run dev
  ```
* **데스크톱 앱 실행 (별도 터미널에서 수행):**
  ```bash
  npm run electron:dev
  ```

### 3. 프로덕션 빌드 및 배포 패키징
프론트엔드 리소스를 컴파일하고 `electron-builder`를 사용하여 배포용 Windows 인스톨러 및 포터블 실행 파일을 생성합니다.
```bash
npm run dist
```
* **출력 폴더:** `dist-electron/` 경로에 `.exe` 설치 파일이 생성됩니다.
* **업데이트 설치 기능:** 사용자의 시스템에 기존 버전이 설치되어 있을 경우, 설치 마법사가 기존 설치본을 감지하여 최신 업데이트 설치를 진행할지 안내 창을 표시합니다.
